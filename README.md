# Сетевые приложения

Система **Ресторан**. **Клиент** осуществляет заказ из **Меню**. **Администратор**
подтверждает **Заказ** и отправляет его на кухню для исполнения.
**Администратор** выставляет **Счет**. **Клиент** производит его оплату.
